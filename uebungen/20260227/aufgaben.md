# Aufgabe 1

Es soll ein Programm für die automatische Berechnung eines Rabatts bei einer Online-Bestellung entwickelt werden. Es gelten folgende Rabattregelungen:
10% Preisnachlass, wenn der Bestellwert 100,00€ übersteigt. 
Ab 200€ Einkauf gibt es 15 % Rabatt.
Erstelle ein Struktogramm und ein PAP.

# Aufgabe 2: Supermarkt-System

Erstelle ein ER-Diagramm für ein Supermarkt-Verwaltungssystem. Die folgenden Anforderungen sollen berücksichtigt werden:
* Ein Kunde kann mehrere Bestellungen tätigen.
* Eine Bestellung kann mehrere Produkte enthalten.
* Jedes Produkt gehört genau zu einer Kategorie (z. B. Getränke, Backwaren).
* Ein Kunde hat Attribute: KundenID, Name, Email.
* Produkte haben Attribute: ProduktID, Preis, Name.

Aufgabe: Modelliere alle Entitäten, Attribute, Beziehungen und Kardinalitäten. Zeichne anschließend ein vollständiges ER-Diagramm.

# Aufgabe 3: Schule / Klassenverwaltung

An einer Schule sollen Informationen digitalisiert werden.
* Eine Klasse hat viele Schüler.
* Jeder Schüler bekommt Zeugnisnoten in mehreren Fächern.
* Ein Fach wird von genau einer Lehrkraft unterrichtet.

Teilaufgaben:
1. Identifiziere alle Entitäten und ordne ihnen passende Attribute zu.
2. Bestimme alle Kardinalitäten.
3. Erstelle das ER-Diagramm in der Crow’s Foot Notation.

# Aufgabe 4: Fitnessstudio-Verwaltung

Ein Fitnessstudio betreibt eine Kundensoftware mit folgenden Anforderungen:
* Ein Mitglied kann mehrere Kurse besuchen.
* Ein Kurs wird von einem Trainer geleitet.
* Ein Trainer kann mehrere Kurse leiten.
* Jedes Mitglied besitzt ein Mitgliedsprofil (ID, Name, Beitrittsdatum).
* Jeder Kurs besitzt Attribute wie KursID, Name, Raum, Zeit.

Modelliere ein ER-Diagramm und stelle sicher, dass alle n:m- und 1:n-Beziehungen korrekt dargestellt werden. Gib die Primärschlüssel der Entitäten an.
